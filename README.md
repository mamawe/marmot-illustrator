# marmot-illustrator

为文章规划并生成可读性强的温暖手绘风配图，以圆胖土拨鼠为固定向导角色。核心差异是「掘进式配图法」——像土拨鼠打洞一样帮文章挖出深层含义，每张图自带环境叙事层（泥土、草地、天空、洞穴等场景暗示）。用户说土拨鼠配图、marmot illustration、给文章画插图、掘进式配图时触发。

## 安装

```bash
npx skills add mamawe/marmot-illustrator --skill marmot-illustrator
```
