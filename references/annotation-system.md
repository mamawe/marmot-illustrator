# 手写标注系统（土拨鼠版大地色系）

文字不是图片标题，而是画面结构的一部分。土拨鼠版的标注使用**大地色系**替代橙猫版的黑白橙色系，整体更温暖。

## 标签的五种作用

1. **命名**：召回、粗排、精排。
2. **路径**：直接生成、回查证据。
3. **问题**：证据不足、假设过多。
4. **结果**：保留、淘汰、推荐。
5. **补充**：冷启动、长尾内容。

每个标签只能承担一种作用。

## 标签规划字段

每个标签在构思卡中记录：

```text
文字：
作用：
颜色：
位置：
是否可以删除：
```

## 标签数量

1. 每张图使用 1 至 5 个标签。
2. 每个标签优先 2 至 4 字，最多 6 字。
3. 不写完整句子。
4. 不放图型标题和口号。

## 标签位置

1. 紧贴对象或路径。
2. 与对象保持稳定间距。
3. 不横跨连接线。
4. 不放在大片留白中央。
5. 必要时用一条短手绘引线。

## 颜色语义（大地色系）

| 颜色 | 用途 | 色彩联想 |
|---|---|---|
| **深棕色** | 对象名称、普通状态 | 泥土、树干、稳重的基础 |
| **森林绿**（帽子的颜色） | 主路径、关键动作、核心节点 | 草地、生机、前进的方向 |
| **暖橙色** | 问题、阻塞、淘汰、风险 | 夕阳、警示、温暖但醒目 |
| **淡蓝色** | 补充状态、外部条件、次要解释 | 天空、远方、开阔 |

**使用规则**：
1. 一张图默认**深棕色**加一种功能色。
2. 只有确实同时存在主路径和问题时，才允许加入第二种功能色（绿+橙）。
3. 绿色应该让人联想到土拨鼠帽子上的那一抹绿——一种亲切的指引色。
4. 暖橙色不应该刺眼，更像夕阳照在泥土上的颜色。
5. 淡蓝色只在天空或远景相关的语境下使用。

## 手写感

1. 字体清楚优先于个性。
2. 轻微不规则，不能潦草。
3. 笔画粗细与主线条协调，略细于模块轮廓。
4. 不使用儿童涂鸦体。
5. 不使用正式黑体或海报标题字。

## 文字 QA

1. 是否逐字正确？
2. 是否足够短？
3. 是否紧贴对应对象？
4. 是否与大地色系语义一致？
5. 删除后是否更清楚？
6. 是否出现图型标题或无意义口号？
7. 手机宽度下仍能辨认？
8. 标签颜色是否和整体大地色系协调（有没有突兀出现的亮红亮紫）？
