# 视觉转译指南（土拨鼠版）

把文字隐喻变成温暖有温度的画面。土拨鼠版的转译原则：**不只传递信息，还要传递感受**。

## 核心区别：从"看懂"到"感受到"

橙猫版的转译目标是让读者**理解**一个判断。
土拨鼠版的转译目标是让读者**感受到**一个判断，同时理解它。

一字之差意味着：
- 不只画出"是什么"，还要暗示"这让我感觉如何"
- 环境层承担了一部分情绪工作，不需要全部堆在主体上
- 允许稍微多一点的"不精确"，如果那种不精确带来了温暖感

## 核心原则：保留一个名词，用温度改造它

文字隐喻由名词构成，情绪藏在质感里。转译时选一个最能代表原文的具体锚点，然后用**掘进四问**改造它：

1. 这个情绪的**重量**是什么样的？（压下来的 / 飘着的 / 沉底的 / 温暖地托着的）
2. 它的**温度**是什么样的？（清晨灰白 / 深夜一盏暖灯 / 正午阳光晒热的石头）
3. 人在这个情绪里的**姿态**是什么样的？（缩着 / 站直但很小 / 走着但回头 / 笑着但眼里有东西）
4. **土拨鼠在这里会做什么？**（刨开看看 / 咬断障碍 / 探头张望 / 安静蹲着陪着你）

第四个问题是土拨鼠版独有的——它强制你把抽象情绪变成一个具体的、有物种特征的行动。

## 三层关联法 + 环境层

一张合格的土拨鼠版插图要同时具备四层：

| 层级 | 回答的问题 | 示例 |
|---|---|---|
| 内容锚点 | 这篇文章具体讲了什么？ | 液态玻璃颜色选择器 |
| 关系结构 | 作者对它做了什么判断？ | 开源模型在具体任务上逼近闭源模型 |
| 情绪转译 | 作者希望读者感到什么？ | 兴奋，但不神化，踏实地看到希望 |
| **环境氛围** | **这个判断发生在什么样的空间里？** | **阳光照在草地上，一只土拨鼠刚从洞里探出头发现新事物** |

前三层和橙猫版相同。第四层是土拨鼠版独有——环境不只是背景，它是情绪的物理化身。

## 简约的纪律

- **概念插图主体不超过两个元素**。一个角色 + 一个物，或一个物 + 一个环境暗示。
- **留白占画面一半以上**。留白是给情绪呼吸的地方。土拨鼠版的留白可以带轻微的环境暗示（比如留白的上方偏蓝表示天空），但不能填充。
- **圆胖土拨鼠是默认角色**。使用固定参考图中的形象。它不是装饰物，动作必须服务于原文。
- **角色宜小不宜大**。默认占画面约 8% 至 12%，比橙猫稍大一点点（因为土拨鼠的圆胖身材本身就有视觉分量）。大部分画面仍然交给原文指纹、关系结构和留白。
- **外形保持稳定**。圆胖身材、棕色背、米黄肚皮、大门牙、绿色毛线帽、大圆眼睛不能漂移。允许改变姿势和表情。
- **线条是主角，但有温度**。使用清晰、有力度且带轻微有机抖动的墨线。不像橙猫版那样追求冷峻的精确感。允许线条有一点"手绘的温暖不完美"。
- **大地色系只做点睛**。以暖棕、米黄和墨绿为主色调，只保留一至两处小面积的强调色。
- **减少装饰纹理**。允许极轻微的自然质感（纸张、泥土），但不能出现大片铅笔排线或密集阴影。
- **领域文章要保留领域证据**。科技文章至少保留一个技术对象，商业文章至少保留一个交易线索，生活文章至少保留一个具体动作或旧物。

## 正反例（土拨鼠版改编）

**例一：文章写"中年转行像船到江心"**

- 直译（差）：一条船在江中央，船上一个人愁眉苦脸。太图解。
- 橙猫版转译（好）：极简水面占满下方三分之一，一个小背影坐在船头，船头朝外，远处空白。悬而未决但朝前的情绪。
- **土拨鼠版转译**：水面仍然是主画面，但在左下角泥土断面处，一只土拨鼠的前爪扒在岸边，眼睛看向水面上那只小船。**环境层**：岸边有几根草叶，泥土有质感。**情绪差异**：橙猫版是"一个人面对不确定"；土拨鼠版是"岸上有人默默为你加油，虽然它帮不了你上船"。多了陪伴的温度。

**例二：文章写"父亲老了，像一盏越来越暗的灯"**

- 橙猫版转译（好）：旧台灯在右下角，光晕很小但是全画唯一的暖色，边缘有一副老花镜。人不出现。
- **土拨鼠版转译**：同样的台灯场景，但在台灯底座旁边，一只小小的土拨鼠安静地蹲坐着，双手抱着膝盖，抬头看着那盏灯。**环境层**：地板木纹，角落有一点灰尘暗示时间的流逝。**情绪差异**：橙猫版是克制和余味；土拨鼠版是"我在这里陪着这盏灯"。更直接的情感连接。

**例三：文章写"AI 时代的焦虑，像所有人都在跑而你在系鞋带"**

- 橙猫版转译（好）：大部分是密集模糊的细线条（人流速度），左下角一个清晰的静止小人蹲着。
- **土拨鼠版转译**：同样的人流模糊线，但左下角是一只土拨鼠——**它没有在系鞋带，而是坐在自己的洞口旁，手里拿着一根鞋带发愣**，周围的人群从它身边跑过。**环境层**：洞口的泥土边缘，一根草叶被跑过的人群踩弯了。**情绪差异**：土拨鼠的洞口是一个安全隐喻——"我可以在我的洞里慢慢来"。把焦虑转化成了一种温和的自洽。

**例四：文章写"一个开源 Coding 模型做出了闭源模型没做好的液态玻璃颜色选择器"**

- 橙猫版转译（好）：代码窗口旁悬浮液态玻璃色盘，远处还有更大的玻璃面板，一近一远表达追赶关系。
- **土拨鼠版转译**：类似的主体布局（代码窗口+玻璃色盘），但增加**环境层**：整个画面像是放在一片阳光照到的草地上，玻璃色盘折射出的彩虹光照亮了旁边一只探头观看的土拨鼠的脸。**掘进深度标记为中层**——泥土草根可见，代表这项技术突破是经过深耕的结果。**情绪差异**：橙猫版是冷静的分析之美；土拨鼠版是"哇这个好厉害"的纯真惊叹。

## 脱文测试

生成前后都问一次：

1. 熟悉文章的人能否看出这是哪一个 Case 或哪一段？
2. 画面里是否有至少一个只能来自这篇文章的线索？
3. 如果把插图挪到另一篇同主题文章里，它是否仍然毫无违和？
4. （土拨鼠版独有）去掉土拨鼠后，环境层是否还能提供足够的情绪温度？

## 几种常见情绪的画面语汇（土拨鼠版）

| 情绪 | 橙猫版倾向 | 土拨鼠版倾向 |
|---|---|---|
| 孤独但平静 | 小人物 + 大留白 | 小人物 + 大留白 + 一只远处静静坐着的土拨鼠（你不是一个人） |
| 压抑 | 上方留白少、线条变密 | 同左 + 泥土层厚重感、光线被遮住 |
| 释然 / 收尾 | 主体面向画面外 | 同左 + 土拨鼠面向同一个方向挥手（送别/祝福） |
| 怀旧 | 旧物特写，线条抖 | 同左 + 土拨鼠用爪子轻轻碰那个旧物（触碰记忆） |
| 笨拙的勇敢 | 人物前倾，环境略大 | 同左 + 土拨鼠在旁边做着同样的笨拙动作（共鸣） |
| 时代感 / 洪流 | 重复线条阵列 | 同左 + 土拨鼠抓着一根固定的小草不被冲走（坚守） |

## 线条检查

1. 缩小后，主体轮廓是否一眼可辨？
2. 轮廓线是否有轻微的有机抖动感？（不应该像尺子画的）
3. 内部线条是否比轮廓更轻更细？
4. 色彩全部去掉后，构图和温度是否仍然成立？
5. 是否出现了过度冷峻的几何感？如果有，加入"organic"、"wobbly"、"warm"等关键词。

## 角色造型检查

1. 是否读取了 `assets/marmot-main-visual.jpg` 作为角色参考？
2. 圆胖体型、棕色皮毛分区、大门牙、绿色帽子和大圆眼睛是否保持稳定？
3. 动作是否表达原文情绪，而且**利用了土拨鼠的物种特征**？（刨、咬、探头、招手、抱住、蹲坐）
4. 是否使用了合适的小道具？（铲子、放大镜、旗子、背包、笔记本）
5. 角色是否控制在约 8% 至 12%？
6. 表情是否丰富自然而不是固定的"笑脸"？

## 环境层检查（土拨鼠版独有）

1. 环境元素是否和掘进深度匹配？
2. 环境元素总面积是否不超过 20%？
3. 去掉环境层后画面是否仍然成立？（环境是加分项，不是必需品）
4. 环境是否提供了正确的情绪温度？
5. 环境元素是否足够具体可画？（不能只是"温暖的氛围"）
