# 配图构思协议（土拨鼠版掘进式）

这份协议负责把文章观点变成读者能感受到的画面。核心隐喻是**掘进**——从地表开始，一层一层往下挖，直到找到文章真正想说的那个东西。

本文件主要负责概念插图和所有图型共有的命题压缩、阅读顺序与验证规则。流程图、架构图等的专门规则见 `diagram-types.md`。

## 一、先决定图片为什么存在

每张图只能有一个主要任务：

1. 解释一个机制。
2. 区分两个概念。
3. 标记一次转折。
4. 让一个案例被记住。
5. 收束一种情绪。

## 二、从主题降到命题

主题无法直接构图。命题必须包含判断或变化：

使用句式：

```
这张图不是要展示______，而是要让读者感受到______。
```

注意"感受到"三个字。土拨鼠版不只追求信息准确，还追求情绪到达。

## 三、掘进式视觉句子

视觉句子由四个部分组成（比橙猫版多一层）：

```
主语 + 变化关系 + 结果 + 环境温度
```

示例：

```
两张诗人内容卡 + 一条表面直线和三条向下根线 + 相似内容与真实动机的分离 + 阳光透过草叶洒在泥土上的温暖感
```

环境温度是土拨鼠版的独有维度。它不是一个独立物件，而是一种**光线+材质+空间的组合感觉**。

## 四、优先寻找主动作（含土拨鼠特色动作）

最容易理解的插图通常是一个动作。通用动作列表：

1. **掀开**，适合表层与深层。（土拨鼠版：**刨开**——用爪子扒开覆盖物）
2. **穿过**，适合阻隔与突破。
3. **筛掉**，适合发散与收敛。
4. **对准**，适合假设与证据匹配。
5. **埋下**，适合积累与生长。
6. **拉近**，适合距离变化和追赶。
7. **接上**，适合断裂关系被重新连接。
8. **回看**，适合验证、修正和复盘。

**土拨鼠专属动作**（充分利用物种特征）：

| 动作 | 适用场景 | 视觉表现 |
|---|---|---|
| **刨开** | 表象 vs 本质 | 前爪扒开草地/泥土，露出下面的东西 |
| **咬断** | 突破限制 | 大门牙啃穿一道屏障/墙/网 |
| **探头** | 发现新事物 | 从洞口探出半个身子张望 |
| **招手** | 引荐/介绍 | 举起一只前爪挥动 |
| **抱住** | 珍惜/保护 | 双臂环抱某个物体 |
| **蹲坐** | 沉思/见证 | 圆滚滚地坐着观察 |
| **竖耳** | 警觉/倾听 | 耳朵竖起来（虽然在毛里若隐若现） |
| **拖拽** | 努力搬运 | 用两只短爪费力地拖某样东西 |

先问土拨鼠在做什么，再问周围需要哪些物件。一个清楚的动作通常只需要一个主物件和一个结果。

## 五、八种关系语法

### 前后变化
新旧交替。画面依靠左右、前后、门槛或跨越表达。

### 表层与深层（土拨鼠版特别擅长）
行为与动机、现象与原因。**天然契合掘进隐喻**——画面可以是地面以上的显性内容和地面以下的隐性根系。土拨鼠站在分界线上。

### 分叉与选择
多种解释或路径。分叉控制在二至四条。

### 发散与收敛
假设筛选。让读者明显看见多变少。

### 阻隔与穿越
限制和新通道。

### 尺度与距离
领先与追赶。不能只画两个大小不同的物体，还要让距离变化服务于判断。

### 积累与生长
长期主义。过程痕迹必须可见。

### 循环与验证
假设-证据-修正循环。

## 六、阅读顺序

1. 第一眼看见唯一主语。
2. 第二眼沿着变化线移动。
3. 第三眼注意到环境层的温度暗示。
4. 最后在结果或对照中感受到文章判断。

## 七、土拨鼠的职责

土拨鼠承担四种职责（比橙猫版多一种）：

1. 操作关键对象（刨开、咬断、拖拽等）。
2. 指示阅读方向（看向分叉、朝向结果）。
3. 提供尺度和情绪（面对巨大系统时显得微小）。
4. **提供情感共鸣**（土拨鼠版独有——通过表情和姿态传递"我和你在一起"的感觉）。

土拨鼠不得：
1. 仅仅站在角落增加可爱感。
2. 同时操作多个对象解释完整流程。
3. 穿复杂服装扮演专业人士。
4. 用夸张表情代替文章判断。
5. **没有任何情绪参与**（土拨鼠版特别禁止——如果移除土拨鼠后画面情绪完全不变，说明土拨鼠只是装饰）。

生成前做**角色必要性测试 + 情感参与测试**：
- 暂时移除土拨鼠。画面含义不变吗？如果是，重新设计动作。
- 土拨鼠的表情和姿态是否传达了某种温度？如果不是，给它一个更有感情的动作。

## 九、构思评分

每项一分，总分至少六分才可生成（比橙猫版高一分，因为多了环境层要求）：

1. 单一命题是否明确。
2. 是否保留独特原文指纹。
3. 是否只有一个视觉主语。
4. 关系变化是否一眼可见。
5. 土拨鼠动作是否参与含义且有物种特色。
6. 是否能删除所有非必要元素。
7. 相似元素是否已压缩到最少。
8. **环境层是否提供了正确的情绪温度而不抢戏。**（土拨鼠版独有项）

总分至少七分才可生成。低于七分时，不写提示词，重新构思。

## 十、掘进式构思示例

### 示例一：相关性 vs 真实动机（表层与深层）

```text
插入位置：
这时候推荐就不只是相似内容的延伸了。

配图任务：区分

单一命题：
相关性只在表面滑行，推理会像土拨鼠一样刨开表层去寻找下面藏着的真实动机。

原文指纹：
李白、杜甫、王维内容卡，以及亲子传统文化、唐代历史、古风创作三种动机。

掘进深度：表层→深层（这张图的核心就是掘进）

视觉主语：
被土拨鼠刨开一角的两张诗人卡片。

关系语法：表层与深层

环境层设计：
- 元素：上半部是草地（地表），下半部是有质感的泥土断面（深层），中间是土拨鼠正在刨的分界线
- 位置：上下各占一半画面
- 面积占比：约 18%
- 光线：从上面照射下来，越往深处越柔和

阅读顺序：
第一眼：土撥鼠正用两只前爪用力刨开草地。
第二眼：草皮下露出一小块泥土，泥土里有不一样颜色的根须。
环境感知：上面的世界明亮轻松，下面的世界厚重但真实。
最后理解：卡片表面看起来差不多，但下面长着完全不同的根。

土拨鼠动作：
两只前爪正在用力向下刨，耳朵因为用力而微微贴后，大门牙露出一点，眼神专注又兴奋。

一秒读图目标：
看出表面下面还藏着东西。

三秒读图目标：
感受到"原来表面相似的下面藏着完全不同的驱动力"。

删除项：
删除放大镜、箭头群、完整流程节点和多余文字标签。

画面一句话：
一只戴绿帽的土拨鼠正在草地上刨坑，爪子下的草皮翻开，露出泥土中两条走向完全不同的根须。
```

### 示例二：假设需要证据帮助收敛（阻隔与穿越）

```text
插入位置：
最后根据验证结果生成推荐。

配图任务：解释

单一命题：
模型不是选择听起来最合理的，而是像土拨鼠咬穿挡路的杂草丛一样，只留下能对准证据的那几个。

原文指纹：
多种兴趣假设、用户行为序列、亲子内容和唐代历史证据。

掘进深度：浅层

视觉主语：
一丛杂乱的假设草，中间有一条被咬开的通路。

关系语法：阻隔与穿越

环境层设计：
- 元素：几根草叶在底部边缘，远处淡蓝天
- 位置：底部和右上角
- 面积占比：约 10%
- 光线：明亮的午后侧光

阅读顺序：
第一眼：土撥鼠对着面前的一丛杂草。
第二眼：杂草被咬出了一个整齐的缺口，缺口后面露出一条清晰的小路。
环境感知：阳光很好，这是一个充满希望的下午。
最后理解：不是所有假设都值得走过去，咬开路障只为了走通的那一条。

土拨鼠动作：
龇着两颗大门牙，正在咬断一根横在前面的粗杂草茎，前爪扶着旁边的草稳住身体。

一秒读图目标：
看出正在清除障碍，不是在吃草。

三秒读图目标：
感受到"筛选不是损失，是为了走通对的那条路"。

删除项：
删除四阶段环形流程、复杂分叉和多个推荐结果卡片。

画面一句话：
戴绿帽的土拨鼠龇着大门牙咬断了一根拦路的粗草，咬开的缺口后面露出一条通向远方的小径，阳光正好照在那条路上。
```
